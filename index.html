Abanico.js

Plugin jQuery para Busquedas Autocompletadas.

Download this project as a .zip file Download this project as a tar.gz file

¿Qué es abanico.js?

Abanico.js es un plugin jQuery, que se complementa perfectamente cuando usamos Bootstrap, básicamente su función consiste en efectuar una búsqueda a través de un input, el cual filtra la información a buscar cada ves que se digita una tecla se despliega un abanico de posibles valores encontrados.

Instalación

Necesitas descargar abanico.js y guardarlo en tu proyecto en la sección js/, luego se debe incluir en el archivo donde se va a usar, de la siguiente forma:

<script src="js/abanico.js"></script>
